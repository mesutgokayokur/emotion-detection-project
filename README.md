# Final Project: Emotion Detection System

## Project Description
This is an AI-powered web application built with Python and Flask. 
It utilizes the IBM Watson NLP library to analyze text provided by users 
and detect various emotions such as joy, anger, fear, sadness, and disgust.

## Features
* Watson NLP integration for emotion analysis.
* Flask-based web server.
* Comprehensive unit testing for analysis logic.
* Error handling for invalid or empty inputs.
* Static code analysis (Pylint) compliant.
